####关于.prettierrc和esLint报error  Insert `·`  prettier/prettier这个问题
```bash
因为prettierrc和esLint两个文件都是对文件格式化的配置文件,假如esLint设置函数前要空格,prettierrc设置不要空格,
这时候就会报上诉的报错,相当于两个配置文件针对同一个问题冲突了,就会导致这个问题,所以只要设置两个文件针对这个问题
都是相同的处理就可以过校验了,也是坑爹
有空查查prettierrc如何设置方法前面去空格的设置
```
